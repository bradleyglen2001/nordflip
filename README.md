NordFlip
Create beautiful split-flap displays in your browser. Inspired by Gare Du Nord station. 

Design custom departure boards, message displays, or decorative boards with the nostalgic aesthetic of mechanical flip displays.

Features

Interactive Designer - Click any row to edit, drag to reorder
Multiple Row Types

Static text displays
Rotating messages with custom intervals
Live time/date displays with timezone support
Visual separators
Two-column layouts


Customizable

Adjustable board dimensions
Custom flip speed and animations
Sound effects toggle
Flexible text alignment


Two Sharing Options

Generate Final - Non-editable shareable display
Preview and Save - Editable link you can bookmark and modify anytime


Responsive - Works on desktop and not yet optimized for mobile browsers

How It Works

Design - Add rows using the + button, click any row to edit content
Customize - Adjust settings in the Advanced panel (grid size, flip speed, etc.)
Share - Generate a final board for sharing, or create an editable link to save and modify later

Use Cases

Airport/train departure boards
Event schedules and announcements
Retro-style message displays
Office dashboards with world clocks
Digital signage with rotating messages
Decorative displays for websites or events
Menus

Technology
Built with vanilla JavaScript, no dependencies required. All processing happens client-side - your designs never leave your browser.
